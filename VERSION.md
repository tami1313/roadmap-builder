# Version Information

## Current Version: 1.0.0

This is the initial stable release of the Roadmap Builder application.

### To revert to this version:

```bash
git checkout v1.0.0
```

### To see what changed since this version:

```bash
git log v1.0.0..HEAD
```

### To create a new branch from this version:

```bash
git checkout -b new-feature-branch v1.0.0
```
