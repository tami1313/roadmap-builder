# Roadmap Builder

A web-based tool for building and managing product roadmaps with a focus on outcomes, problems to solve, and validation needs.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Phase 1**: Define high-level outcomes (North Stars) with timeline spanning
- **Phase 2**: Add specific problems to solve under each outcome
- Validation tracking (pre-build and post-build)
- External roadmap view with Instructure-style collapsible sections
- Local storage persistence
